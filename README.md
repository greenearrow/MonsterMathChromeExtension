# MonsterMathChromeExtension
See https://developer.chrome.com/docs/extensions/mv3/getstarted/development-basics/#load-unpacked to set up extensions after downloading git repo

This primarily allows you to click through D&D Beyond pages and it will automatically download the page. Monsters, spells, and magic-items will only download 1x (per reset, stored in Chrome local storage). Source books will currently download every time.
